# Нутрициолог Атакова Лада — Сайт и PWA-приложение

Сайт и PWA-приложение для нутрициолога Атаковой Лады. Специализация — раздельное питание.

## 🚀 Возможности

- **Лендинг** — продающая страница с информацией о нутрициологе и тарифах
- **PWA-приложение** — установка на телефон как веб-приложение
- **Отчёты о питании** — завтрак, обед, полдник, ужин, внеплановое питание + фото
- **Замеры тела** — вес, объёмы с графиками трендов
- **Трекер воды** — учёт потребления воды
- **Чат с куратором** — обратная связь и рекомендации
- **Панель куратора** — мониторинг результатов клиентов
